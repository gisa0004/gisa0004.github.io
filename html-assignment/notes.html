During the previouse 4 weeks, here are some of the tags that i have learnt;
  • <p></P>:
    • this tag is use when writting a paragraph. these could be separated by numbering them like p1,p2,p3 to show that there is a new start on the paragraph.
  • <h1></h1>:
    • This tag is used to insert a heading for a page.
  • <cite></cite>:
    • Thistag is used when giving a citation for a book,movie,URL,etc.
  • <blockquote></blockquote>:
    • This is a large stand alone quote.This is commonly used when pages are using famous quotes.
  • <q></q>:
    • This tag is used to represent quotes embedded within the content.
